## Installation

**Prerequisites:**
- NodeJS
- NPM
- MongoDB
- Redis

**Installation steps:**

### Option 01: docker-compose
- `cp .env.example .env`
- fill the .env file in with proper credentials for docker environment
- `docker-compose up`

### Option 02: Regular installation
- `npm install`
- `cp .env.example .env`
- fill the .env file in with proper credentials
- `npm start`
